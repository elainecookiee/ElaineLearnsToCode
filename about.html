ABOUT

Today is December 19, 2015 and the start of the “Elaine Learns to Code Project”. Similar to Jennifer Dewalt’s project, I will be publishing all my code as an open source on GitHub. Instead of having one blog post to accompany each site, I am writing a blog post each day about my thought and implementation process from brainstorming ideas, sketching, wire framing to the resources that I find are actually useful.

Background:

  • New grad from UW, Honours Psychology Spring of 2015.
  • Self taught HTML, CSS and Python (beginner level).
  • No experience with using web design tools or programming.

What I want to get out of this:

  • The best practice I could ever get.
  • A hand full of tools that would help me plan, design and run a website.
  • Create interactive sites and become familiar with Javascript, JQuery and other programming languages.
  • Explore design away from traditional illustration.

I am interested in:

  • UX/UI design, Graphics Design, Web Development, Psychology, Illustration, Painting, Photography
